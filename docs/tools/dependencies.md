# Dependencies & SDKs

This document lists the libraries and SDKs commonly used in Zotero plugin development.

## [zotero-plugin-toolkit](https://github.com/windingwind/zotero-plugin-toolkit)

<div style="display: flex; gap: 10px;">
  <img src="https://img.shields.io/npm/v/zotero-plugin-toolkit?style=flat" alt="version" />
  <img src="https://img.shields.io/npm/dt/zotero-plugin-toolkit?style=flat" alt="downloads" />
  <img src="https://img.shields.io/github/stars/windingwind/zotero-plugin-toolkit?style=flat" alt="start" />
</div>

A comprehensive toolkit that simplifies the development of Zotero plugins. It includes utilities and helpers for common tasks such as menu management, preference handling, and UI component creation. Some key features are gradually being integrated into the official Zotero plugin API.

This library is by default included when you use the [windingwind/zotero-plugin-template](https://github.com/windingwind/zotero-plugin-template).

## [zotero-types](https://www.npmjs.com/package/zotero-types)

<div style="display: flex; gap: 10px;">
  <img src="https://img.shields.io/npm/v/zotero-types?style=flat" alt="version" />
  <img src="https://img.shields.io/npm/dt/zotero-types?style=flat" alt="downloads" />
  <img src="https://img.shields.io/github/stars/windingwind/zotero-types?style=flat" alt="start" />
</div>

TypeScript type definitions for Zotero's XPCOM interfaces and APIs. This package provides type safety and autocompletion support when developing Zotero plugins using TypeScript.

This library is by default included when you use the [windingwind/zotero-plugin-template](https://github.com/windingwind/zotero-plugin-template).
